Front End Developer

Front end developers are programmers who work on the 'front end' or user interface of a website, they will primarily code in languages such as HTML, CSS and Javascript. A rough salary estimate for a front end developer is around £55,000

Back End Developer

Back end developers are programmers who work on the server side of things which will involve coding in languages such as PHP, SQL and even languages such as Python and C#. A rough salary estimate for a back end developer is around £64,000

Systems Architect

Systems architects are the people who are resposible for building and configuring the network infrastructure and servers of which the website will run and rely on. Languages a systems architect should know consist of are Python, Java, C, C++ and C#. A rough salary estimate for a systems architect is around £76,000
